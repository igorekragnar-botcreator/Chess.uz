# Chess.uz
## 🎮 Возможности
 
- **2 режима игры** — два игрока на одном устройстве или против AI
- **6 уровней сложности** — от новичка до гроссмейстера (~2600 Elo)
- **Книга дебютов Карлсена** — 300+ позиций из реального репертуара Магнуса Карлсена
- **Определитель дебютов** — автоматически показывает название дебюта в реальном времени
- **Анимация ходов** — плавное движение фигур
- **Таймер** — 10 минут на каждого игрока
- **История ходов** — с кнопкой копирования в PGN формат
- **Захваченные фигуры** и оценочная шкала материала
- **Flip доски** — перевернуть доску
- **Промоция** — выбор фигуры при превращении пешки
- **Звуки** — ходы, взятия, рокировка, шах
- **Адаптивный дизайн** — работает на телефоне и ПК

## 🧠 AI Движок
 
Движок написан полностью на JavaScript. Использует:
 
| Техника | Описание |
|---|---|
| **Minimax + Alpha-Beta** | Основной алгоритм поиска |
| **Iterative Deepening** | Поиск от глубины 1 до максимума |
| **Transposition Table** | Кэш 512K позиций, ускорение ×3-5 |
| **Quiescence Search** | Продолжает поиск в тактических позициях |
| **Null-Move Pruning** | Отсечение явно плохих ветвей |
| **Late Move Reduction (LMR)** | Поздние ходы ищутся на меньшую глубину |
| **PVS (Principal Variation Search)** | Узкое окно для быстрого отсечения |
| **Killer Heuristic** | Запоминает лучшие тихие ходы |
| **MVV-LVA сортировка** | Сначала выгодные взятия |
| **Книга дебютов** | 300+ позиций из репертуара Карлсена |
| **Оценочная функция** | Пешечная структура, безопасность короля, открытые линии, проходные пешки, фаза игры |

## 📖 Книга дебютов Карлсена
 
AI играет по реальному репертуару Магнуса Карлсена, основанному на:
 
- Матчах за звание чемпиона мира 2013, 2014, 2016, 2018, 2021
- Турнирах Norway Chess, Tata Steel 2019–2023
**За белых:** Испанская (Берлин, Морфи), Итальянская, Шотландская, Каталонское начало, Ферзевый гамбит, Нимцо-Индийская, Лондонская система, Английское начало
 
**За чёрных:** Берлинская защита, Свешников, QGD, Нимцо-Индийская, Ферзевая Индийская
